# Dashboard

## Objetivo

El Dashboard es el Centro de Operaciones de Forge OS.

Es la primera pantalla que visualiza el usuario al iniciar sesión.

Su propósito es ofrecer una visión completa del estado de la empresa y facilitar la toma de decisiones.

---

# Objetivos

- Mostrar información crítica.
- Priorizar tareas importantes.
- Reducir el tiempo necesario para encontrar información.
- Centralizar la actividad diaria.

---

# Componentes

- Saludo personalizado.
- Fecha.
- KPIs.
- Focus del día.
- Agenda.
- Actividad reciente.
- Panel de Auren AI.

---

# KPIs

- Empresas activas.
- Discoveries pendientes.
- Proyectos activos.
- Ingresos mensuales.
- Tareas pendientes.

---

# Focus

El Dashboard debe destacar una única tarea prioritaria.

El usuario debe saber inmediatamente cuál es la acción más importante del día.

---

# Auren AI

Auren analiza el sistema y propone acciones estratégicas.

Nunca muestra información irrelevante.

Siempre prioriza el impacto.

---

# Principio

El Dashboard no informa.

El Dashboard dirige.
