# Empresas

## Objetivo

Administrar todas las empresas registradas dentro de Forge OS.

Cada empresa representa un Workspace independiente.

---

# Información Principal

- Nombre
- Industria
- Contacto principal
- Estado
- Forge Score
- Próxima reunión

---

# Funciones

- Crear empresa.
- Editar empresa.
- Eliminar empresa.
- Buscar empresa.
- Filtrar empresas.
- Abrir Workspace.

---

# Principio

Las empresas son el núcleo del sistema.

Todo gira alrededor de ellas.
