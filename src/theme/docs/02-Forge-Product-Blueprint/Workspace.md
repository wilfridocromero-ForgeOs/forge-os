# Workspace

## Objetivo

Centralizar toda la información relacionada con una empresa.

Cada Workspace representa la operación completa de un cliente.

---

# Secciones

Dashboard

Discovery

Forge Score

Plan Estratégico

Proyectos

Documentos

Notas

Calendario

Resultados

Auren AI

---

# Filosofía

Toda la información debe encontrarse dentro del Workspace.

Nunca distribuir información importante entre diferentes módulos.
